# SF6 OKI LOL


## Development

1. Make sure you have Node/NPM installed. (can be downloaded [here](https://nodejs.org/en/download))
2. Clone the repository from https://github.com/tcd/OKI
3. Run the following commands at the root of the project:
    - `npm install`
    - `npm start`
